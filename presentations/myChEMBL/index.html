  • Little to no configuration required.
  • Can be run on standard PC, server, cloud.
  • Saves time and effort.

Installing using Vagrant

  • Install VirtualBox
  • Install Vagrant
  • In the terminal, run: 
    
vagrant init chembl/mychembl_20_ubuntu
vagrant up

    or:

    
vagrant init chembl/mychembl_20_centos
vagrant up
  • Done!

Installing using Docker

  • Requires 64-bit Linux box with modern kernel
  • Smaller download
  • Faster execution
